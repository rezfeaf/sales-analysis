# Sales Data Analysis

A Python data analysis project exploring sales trends from a real-world superstore dataset.

## Questions Answered
- Which region has the highest sales?
- Which category makes the most money?
- What are the top 10 best selling products?
- Which months have the highest sales?

## Libraries Used
- Pandas
- Matplotlib

## How to Run
1. Install libraries: `pip install pandas matplotlib`
2. Add the dataset CSV file to the project folder
3. Run: `python analysis.py`
4. Charts will be saved as PNG files in the project folder

## Dataset
Superstore Sales Dataset from Kaggle
